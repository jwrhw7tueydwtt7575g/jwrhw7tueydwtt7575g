## 🚀 Featured Products & Innovations

### 🔐 Agent Execution Ledger
> **Real-time observability, tamper-evidence & causal attribution for autonomous AI pipelines**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) ![Python 3.12](https://img.shields.io/badge/Python-3.12-blue) ![FastAPI](https://img.shields.io/badge/FastAPI-005571?logo=fastapi) ![React](https://img.shields.io/badge/React-%2320232a.svg?logo=react) ![Docker](https://img.shields.io/badge/Docker-%230db7ed.svg?logo=docker)

An independent observer layer that cryptographically tracks every tool call, output, and agent handoff in multi-step LLM pipelines — solving the **Cryptographic Wall Problem**: silent tool failures that cause LLMs to hallucinate confident but incorrect answers.

**Core Innovations:**
- 🔗 **Three-Layer Cryptographic Hashing** — SHA-256 of inputs, outputs, and blockchain-style chain hashes to prevent tampered replays
- 📉 **Semantic Drift Engine** — Real-time cosine distance scoring between actual tool output and the agent's interpretation using `all-MiniLM-L6-v2`
- 🎯 **Causal Blame Attribution** — Counterfactual DAG that isolates the exact step triggering a cascading pipeline failure
- 🕵️ **Anomaly Inspector** — Detects F1 (Silent API Failure), F3 (Ghost Call), F6 (Confidence Inflation), F8 (Permission Violation) outside LLM contexts
- 🔌 **Zero-Intrusion Adapters** — Wraps LangChain, CrewAI, AutoGen tools in 1 line; business logic untouched

**Stack:** Python · FastAPI · MongoDB Atlas · React · Vite · WebSockets · Recharts · React Flow · Docker

### 🌌 Aether Flow — Enterprise AI Data Pipeline
> **11-stage autonomous pipeline: raw datasets → executive-level intelligence**

![Azure](https://img.shields.io/badge/Azure_Synapse-%230072C6.svg?logo=microsoftazure&logoColor=white) ![GPT-4o](https://img.shields.io/badge/GPT--4o-412991?logo=openai&logoColor=white) ![Next.js](https://img.shields.io/badge/Next.js-black?logo=next.js&logoColor=white)

A state-of-the-art autonomous data pipeline powered by **Azure Synapse Analytics** and **GPT-4o**, with a fully dynamic execution layer that adapts to any schema in real time — no hardcoded steps.

**Core Innovations:**
- 🧠 **LLM Execution Planning** — GPT-4o analyzes statistical digests and generates a custom cleaning + enrichment plan per dataset
- 📊 **Statistical Reliability Layer** — IQR-based anomaly detection and row-level quality indexing
- 🤖 **Data Intelligence Agent** — Conversational NLQ agent grounded in full pipeline state, enrichment justifications, and discovered patterns
- 💡 **Dynamic NLQ Suggestions** — UI queries auto-generated from discovered data issues (nulls, high-variance fields)
- 🖥️ **Executive Observability Dashboard** — Premium Next.js UI with real-time audit trails and signal-vs-noise analysis

**11-Stage Architecture:**
`Ingestion` → `Schema Normalization` → `Delta Lake` → `Statistical Digest` → `LLM Execution Plan` → `Spark Cleaning` → `AI Enrichment (NER · Sentiment · Urgency)` → `Pattern Discovery` → `Insight Synthesis` → `Intelligence Agent` → `Interactive Viz`

**Stack:** Azure Synapse · GPT-4o · Apache Spark · Isolation Forest · Next.js · Delta Lake
